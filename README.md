## Build Setup

``` bash
# install dependencies 首先安装依赖项
npm install

# serve with hot reload at localhost:8080 然后，开发环境，在本地主机上热重载服务:8080
npm run dev

# build for production with minification 生产环境，上线打包
npm run build
```
